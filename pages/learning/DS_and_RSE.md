Data Science and Software Engineering in Manufacturing
([MAC4112](https://sheffield.ac.uk/made4manufacturing/modules/year-1-modules#MAC4112DataScienceandSoftwareEngineeringinManufacturing))
is a PhD level academic module offered by the [EPSRC Centre for Doctoral Training in Machining, Assembly and Digital
Engineering for Manufacturing (MADE4Manufacturing)](https://sheffield.ac.uk/made4manufacturing), a collaboration between
the Faculty of Engineering at the University of Sheffield and the [Advanced Manufacturing Research Centre
(AMRC)](https://www.amrc.co.uk/). [Romain Thomas](/contact/romain-thomas/) and [Tamora James](/contact/tamora-james/)
are module co-leads responsible for the delivery of the Software Engineering sections of the module.


This module introduces the skills and knowledge to apply state-of-the-art data science and software engineering techniques
in a manufacturing context.  Alongside key data science techniques, students learn about the software engineering tools
that are needed to apply those techniques, for example software and data management, version control, code design and
configuration, testing, and reproducible software.

Topics:

- Introduction to the FAIR principles
- Software lifecycle planning
- Version control with git and GitHub
- Code design
- Reproducible computational environments
- Software testing and continuous integration
- Documentation

Research students and staff who are interested in these topics can register for the sessions delivered as part of the
[FAIR² for Research Software](/learning/fair4rs/) programme.

For further information please contact Tamora James ([t.d.james@sheffield.ac.uk](mailto:t.d.james@sheffield.ac.uk),
Programme manager), and/or Romain Thomas, ([romain.thomas@sheffield.ac.uk](mailto:romain.thomas@sheffield.ac.uk), Head
of the RSE group).
